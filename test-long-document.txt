AI 文档摘要系统测试文档

一、项目概述
本项目是一个基于 Next.js 和智谱 AI 的智能文档摘要系统。系统能够自动分析上传的文档内容，并生成简洁准确的摘要，帮助用户快速了解文档核心内容。

二、主要功能
1. 文档上传功能
   - 支持多种文本格式（.txt, .md 等）
   - 基于 Supabase 对象存储
   - 提供拖拽上传体验

2. 文档管理功能
   - 查看已上传文档列表
   - 显示文件详细信息（大小、上传时间）
   - 支持文档删除操作

3. AI 智能摘要
   - 基于智谱 AI GLM-4 模型
   - 自动生成文档摘要
   - 支持中文和英文内容
   - 最大支持 10 万字符（超长自动截断）

三、技术栈
1. 前端框架：Next.js 16 + React 19
2. 编程语言：TypeScript
3. 样式方案：内联样式 + 响应式设计
4. 后端服务：
   - Supabase：对象存储和数据库
   - 智谱 AI：自然语言处理和文档摘要
5. 部署平台：Vercel

四、系统架构
采用前后端分离的架构设计：
- 前端：React 组件 + Next.js App Router
- API 层：Next.js API Routes
- 存储层：Supabase Object Storage
- AI 层：智谱 AI API

五、核心特性
1. 响应式设计
   - 支持桌面端和移动端
   - 自适应布局
   - 良好的用户体验

2. 错误处理
   - 文件大小检查
   - 内容格式验证
   - 友好的错误提示
   - 超长文本自动截断

3. 性能优化
   - 使用 GLM-4-Flash 模型，响应速度快
   - 异步加载文件列表
   - 智能缓存机制

六、安全性
1. API 密钥安全
   - 环境变量管理
   - 不提交到 Git 仓库
   - Vercel 环境变量加密存储

2. 文件安全
   - 文件类型验证
   - 大小限制
   - 安全的文件名处理
